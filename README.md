# FillHome

[![IMAGE ALT TEXT HERE](https://jphacks.com/wp-content/uploads/2023/07/JPHACKS2023_ogp.png)](https://www.youtube.com/watch?v=yYRQEdfGjEg)

## 製品概要
遠方から引っ越す際に、内見したい物件が複数ある人が内見のために引っ越し先に赴く際、候補の物件に泊まることができるWebアプリです。

### 背景(製品開発のきっかけ、課題等）
#### きっかけ
去年から深夜の隣人の騒音に悩まされ引っ越すことになりました。次の家を探すときに騒音がないという条件が生まれたのですが、自分や他の住人が家にいるのは夜であるため、夜の状況を知りたいはずであるのに、内見できるのは昼間でしかなく結局、夜の様子はわからず不安を抱えて入居してしまいました。そこで、Airbnbのように自分が借りたい物件に泊まれて、夜の状況を知ることができるサービスが欲しいと思い着想しました。

#### 課題
##### 賃借人
* 不動産屋の時間制限により、夜の物件の様子を知ることができない
* 内見中にゆっくり駅・スーパーとの距離、町の雰囲気をじっくり知る時間を取ることができない
* 遠方からの内見の場合、必ず宿を取る必要がある
##### 管理会社・大家
* 空室がありつつもそれを利用して収入を得る方法が確立していない
* 物件と賃借人のミスマッチにより、賃借人がすぐ引っ越した場合、手続きを含めとても大変

#### 社会的背景
##### 1. シェアリングサービス・民泊の普及
現在、Airbnbの拡大、民泊の普及によって若い世代を中心に人の物件に安く泊まるという概念が普及しつつあります。2018年には新民泊法が施行されより民泊をしやすくなりました。今回提案するFillHomeは、この新民泊法における「年間提供日数の上限は180日」・「入居者の募集が行われている家屋」という条件を満たしやすく向いていると考えています。
##### 2. コロナ禍によるテレワークの普及による住む場所の重要性の向上
コロナ禍によるテレワークの増加により、家で仕事するために住居へのこだわりが高まっていることが考えられます。この変化の中、これから自分が借りる予定の物件への最後の決め手として、内見の時間制約に縛られず、実際に住んでみて雰囲気や住む環境、詳細な家の構造を知ることで、納得感を持って家を選べることはできるのはないかと考えました。


### 製品説明（具体的な製品の説明）
### 特徴
#### 1. 予約者が内見する物件に泊まることができる
従来の内見では、不動産屋が同伴することにより営業時間の関係で内見の時間帯が昼間に制限されていたり、スーパーや駅までの距離など周辺の状況を見るための行動などが躊躇されていると考えます。
FillHomeを使えば、自分でチェックイン・チェックアウトができる。

#### 2. 予約者が実質無料で泊まることができる
宿泊にお金がかかるという当たり前に対して、仲介手数料に宿泊費が含まれることで、実質無料で泊まることができるという体験を届けます。また、宿泊費を根本的な収益にしていないため価格設定を安くでき他のホテルに対して優位性が保つことができます。

#### 3. 大家が保有物件の空室を収入源にできる
ビルを壊し土地を売った際に一定期間駐車場として貸し出すことで収入を得るのは当然になっているが、賃貸マンションは空室期間中、収入を得ることが難しいという現状に対し、FillHomeを使えば大家や管理会社が空室を収入源にできると考えられます。

### 解決出来ること
#### 1. 自分の都合に合わして自由に内見できる
物件に泊まれることで、誰にも時間に縛られず、予約者は実際家に滞在する夜の時間の物件含め周辺環境や、駅・スーパーなどの周辺施設を自分が満足するまで確認することができます。

#### 2. 実質無料で宿泊できる
遠方から内見をする予約者は、内見が1日作業になるため必ず宿を引っ越し先に取る必要があります。その際の宿はこれから住む場所であるため豪華なホテルではなく、なるべく費用を抑えた宿に泊まりたいのではないかと考えております。このような方に、FillHomeを使っていただくことで、契約が完了すれば仲介手数料に宿泊費が含まれるため、実質無料で宿泊が可能となり無料で宿に泊まることができます。

#### 3. 賃貸者の早期引っ越しを防ぐことができる
実際に物件に泊まることで、賃貸者と物件のミスマッチを防ぐことがより可能となり、早期の引っ越しを防ぐことができる。これにより大家・管理会社の負担は少なくなると考えられます。

#### 4. 大家が空室を収入源にできる
特徴にも書いたが大家が空室を収入源にできていない。また、日本では住宅過剰社会になっており、今後賃貸における空室も問題になってくると考えられます。その現状に対して、FillHomeを使うことで大家が空室を収入源にできると考えています。
